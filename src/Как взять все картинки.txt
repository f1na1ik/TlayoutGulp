БЫСТРО ВЗЯТЬ КАРТИНКИ С МАКЕТА:

1. Тыкаем на все картинки с помощью альта по одной
2. переименовываем их в name.png
3. Нажимаем ФАЙЛ => ГЕНЕРИРОВАТЬ => НАБОР ИЗОБРАЖЕНИЙ.
